Kiểm tra

```
python3 --version
```

Cài đặt `pip`

```
sudo apt install python3-pip
```

Kiểm tra `pip`

```
pip3 --version
```

Thiết lập `python` trỏ tới `python3`

```
echo "alias python='python3'" >> ~/.bashrc
source ~/.bashrc
```
